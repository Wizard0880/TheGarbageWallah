# GarbageWallah

GarbageWallah is a React application built with Vite that aims to streamline the process of waste collection by connecting residents with waste collectors (kabadiwalas). This project provides a user-friendly interface to book pick-up services for scrap and waste.

## Features

- **Navbar**: A fixed navigation bar with links to different sections.
- **Landing Page**: A welcoming page that highlights the purpose of GarbageWallah.
- **Marquee**: A scrolling marquee showcasing the GarbageWallah brand.
- **About**: Information about the GarbageWallah project and its mission.
- **Eyes**: An interactive section with eyes following the cursor.
- **Featured Services**: Display of key services offered by GarbageWallah.

## Components

- `App`: The main component that assembles all the other components.
- `Navbar`: The navigation bar component.
- `LandingPage`: The landing page component.
- `Marquee`: The scrolling marquee component.
- `About`: The about section component.
- `Eyes`: The interactive eyes component.
- `Featured`: The featured services component.
- `InformationSection`: A section providing detailed information and social media links.

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/Wizard0880/TheGarbageWallah.git

2. Navigate to the project directory:
    cd TheGarbageWallah

3. Install the dependencies :
    npm install

## Usage
1. Start the development server :
    npm run dev
2. Open your browser and go to http://localhost:3000 to view the application.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A fast build tool and development server.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Framer Motion**: A library for animations and gestures.
- **React Icons**: A collection of popular icons.
